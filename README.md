# Summerville Credit Union — nFinia Digital Banking Guide

Welcome to the Summerville Credit Union product documentation for the nFinia Digital Banking platform. This guide provides a comprehensive walkthrough of every feature available to members through the Summerville CU web and mobile banking experience.

## How to Use This Guide

This documentation is organized to follow the natural member journey — from first login through everyday banking tasks. Each section covers a major functional area of the platform, with detailed feature guides inside.

| Section                         | What You'll Find                                                              |
| ------------------------------- | ----------------------------------------------------------------------------- |
| **Authentication & Onboarding** | Login, MFA, biometrics, device management, and delegated access               |
| **Accounts & Dashboard**        | Dashboard overview, account details, financial tools, and profile management  |
| **Transfers & Payments**        | Internal transfers, ACH, wire, FedNow, check services, and recipient setup   |
| **Bill Pay, Loans & Cards**     | Bill pay, loan payments, full card lifecycle, and card controls               |
| **P2P & Zelle**                 | Person-to-person payments via Zelle integration                               |
| **Business Banking**            | Commercial hub, ACH/wire operations, role and user management, approvals      |
| **Settings & Preferences**      | Alerts, notifications, personal info, eStatements, and app preferences        |

## About This Platform

Summerville Credit Union's digital banking is powered by the **nFinia** platform from Tyfone. nFinia delivers a modern, responsive banking experience across web browsers, iOS, and Android — with a unified feature set across all channels.
